# Agentic Trip Planning Software

## Rough steps
User creates an account
Clicks 'Plan New Trip'
Enters a destination and dates 
	- This can either be a city or a country
	- If this is a country, it plans which city(s) to visit, and finds accomodation/transport between
Describes or clicks categories for what they want their trip to include (potential pintrest integration)
Enters general preferences or restrictions (e.g. vegetarian, halal, allergies, etc)
Clicks 'Start Planning'
Agents with specific roles complete various trip planning activities
Returns itineraries, flight booking links, accomodation booking links, links to tickets if required by specific attactions. 

Should have a flight managment page
Should have an accomodation management page
Must have a daily itinerary management page
Should have the ability to delay an itinerary item to another day

As someone who does not take trips often
I want to create a trip planning tool
So that I can:
- Maximise time spent well at my trip destinations
- See more attractions at each destination
- Avoid tourist traps
- Find cheap hotels and flights
